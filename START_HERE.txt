🚀 DÉMARRAGE EN 30 SECONDES
══════════════════════════════════════════════════════════════

Étape 1: Lancez le serveur
   Ouvrez un terminal dans ce dossier et tapez:

   (ou: npx http-server)

Étape 2: Ouvrez dans votre navigateur
   
   http://localhost:8000
   
   Vous devriez voir une page immersive avec animations 3D!

✨ FONCTIONNALITÉS AVANCÉES
══════════════════════════════════════════════════════════════

✅ Debug Mode
   Appuyez: Ctrl+Shift+D
   Ou tapez: app.toggleDebugMode(true)

✅ Commandes Console
   showStats()           → Afficher les performances
   goto('section-3')     → Naviguer vers une section
   window.scene3D        → Accès à la scène 3D
   window.app            → Instance de l'app

✅ Configuration
   Modifiez CONFIG dans assets/js/config.js
   Personnalisez les couleurs, animations, etc.

✅ Performance
   Metrics disponibles via showStats()
   Monitoring intégré dans Chrome DevTools


🎓 EXEMPLES D'UTILISATION
══════════════════════════════════════════════════════════════

AJOUTER UNE ANIMATION:
   Ajoutez la classe dans index.html:
   <div class="fade-in-up">Contenu animé</div>

AJOUTER DU PARALLAX:
   <h2 data-parallax="0.5">Titre parallax</h2>

CRÉER UNE NOUVELLE SECTION:
   <section class="section">
       <h2 class="section-title">Ma Section</h2>
       <p>Mon contenu...</p>
   </section>

AJOUTER UN OBJET 3D:
   Modifiez 3d-scene.js → createGeometry()

DÉPLOYER EN LIGNE:
   Consultez DEPLOYMENT.md pour 6 options
