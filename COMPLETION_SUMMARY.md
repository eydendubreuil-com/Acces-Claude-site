## 🚀 Démarrage Rapide

### 1. Lancer le serveur
```bash
cd /workspaces/Acces-Claude-site
python -m http.server 8000
```

### 2. Ouvrir dans le navigateur
```
http://localhost:8000
```

### 3. Personnaliser
- Modifiez le contenu dans `index.html`
- Ajoutez vos styles dans `assets/css/`
- Créez vos animations dans `assets/js/`
