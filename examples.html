Exemples Avancés

Découvrez comment utiliser les fonctionnalités avancées de l'expérience immersive.

1. Animations Personnalisées au Scroll

Vous pouvez créer des animations complexes qui réagissent au scroll:

// Créer un effet de parallax personnalisé ScrollEffectManager.createScrollParallax(element, 0.5); // Créer un effet de fade au scroll ScrollEffectManager.createScrollFade(element); // Créer un effet de scale au scroll ScrollEffectManager.createScrollScale(element, 0.8, 1.2); // Créer un effet de rotation au scroll ScrollEffectManager.createScrollRotate(element, 180);

2. Gestion du Parallax Dynamique

Ajoutez et retirez des éléments parallax dynamiquement:

// Ajouter un nouvel élément parallax parallaxManager.addElement(document.querySelector('.mon-element'), 0.5); // Modifier la force du parallax parallaxManager.setParallaxStrength(25); // Retirer un élément parallax parallaxManager.removeElement(document.querySelector('.mon-element'));

3. Ajouter des Objets 3D

Créez et ajoutez des géométries 3D personnalisées:

// Créer une géométrie personnalisée const geometry = new THREE.BoxGeometry(15, 15, 15); const material = new THREE.MeshPhongMaterial({ color: 0xff00ff, emissive: 0x660066 }); // Ajouter à la scène scene3D.addObject(geometry, material, new THREE.Vector3(0, 0, 0)); // Créer un ripple effect au clic scene3D.createRipple(new THREE.Vector3(0, 0, 0));

4. Contrôle de la Scène 3D

Contrôlez l'animation 3D en temps réel:

// Mettre en pause la scène scene3D.pause(); // Reprendre la scène scene3D.resume(); // Changer la couleur de fond scene3D.changeBackgroundColor(0xff0000); // Retirer un objet scene3D.removeObject(monObjet);

5. Événements Personnalisés

Écoutez les événements de révélation d'éléments:

// Écouter quand un élément est révélé document.addEventListener('reveal', (e) => { console.log('Élément révélé:', e.detail); }); // Ajouter une classe avec délai Utils.addClassWithDelay(element, 'my-class', 500); // Lancer une animation de valeur Utils.animateValue(0, 100, 1000, (value) => { console.log('Valeur actuelle:', value); });

6. Utilitaires Avancés

Utilisez les fonctions utilitaires pour des calculs avancés:

// Normaliser une valeur const normalized = Utils.normalize(50, 0, 100); // 0.5 // Interpolation linéaire (lerp) const interpolated = Utils.lerp(0, 100, 0.5); // 50 // Limiter une valeur const clamped = Utils.clamp(150, 0, 100); // 100 // Obtenir les dimensions du viewport const viewport = Utils.getViewportDimensions(); // Calculer la direction du mouvement const direction = Utils.getMouseDirection(0, 0, 10, 10); // Throttle une fonction const throttled = Utils.throttle(() => console.log('Throttled'), 1000); window.addEventListener('scroll', throttled);

7. Configuration Avancée

Personnalisez les paramètres globaux:

// Accédez à la configuration console.log(window.app.getConfig()); // Mettez à jour la configuration window.app.updateConfig({ parallaxStrength: 30 }); // Activez le debug mode window.app.toggleDebugMode(true); // Afficher les statistiques de performance window.showStats(); // Naviguer vers une section window.goto('section-5');
